# 图像处理与机器视觉实验

本仓库包含了本课程涉及的所有各章实验项目，主要结构及实验内容如下：

- **实验1**：图像基础操作。涵盖图像的基本读取、显示等环境操作准备。
- **实验2**：图像空间域增强。分为图像平滑（降噪）、图像锐化（边缘强化）以及图像代数运算三个任务。
- **实验3**：图像频域滤波处理。主要进行傅里叶变换及相关频域滤波（如低通、高通滤波）操作。
- **实验4**：数学形态学处理。涉及包含腐蚀、膨胀、开闭运算在内的基础形态学操作。
- **实验5**：图像特征提取。提取并评估图像的关键特征（如边缘等）。
- **实验6**：图像分割。结合传统经典的图像分割方法以及基于 YOLOv8 (yolov8n-seg) 的实例分割机制。
- **实验7**：车牌识别技术应用。综合运用前期所学的图像特征提取、形态学处理等实现车牌定位及识别任务。
- **实验8**：基于 YOLOv8 的目标检测。完整体验深度学习目标检测流程，包含视频抽帧提取（视频转图像集）、自定义数据集构建、模型训练及结果预测。
